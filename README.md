# Seeing School Differently

This repository was developed for **GEOG 521: Critical GIS** at the University of Washington. The course is organized around Critical GIS, participatory mapping, counter-cartography, geospatial storytelling, and the politics of spatial data. The course repository and syllabus are available at [jakobzhao/geog521](https://github.com/jakobzhao/geog521).

## Project Overview

**Seeing School Differently** is a web-based Critical GIS project by **Sechang Kim, UW Geography PhD Student**. The site explores how geospatial methods can represent not only locations and distributions, but also lived experience, meaning, emotion, belonging, and exclusion.

The project began from a proposal on students' everyday school geographies and expanded into several web modules that practice different forms of data collection, classification, visualization, and critical interpretation.

## What This Repository Contains

- `index.html` introduces the main project: a critical qualitative GIS approach to students' school spaces, meanings, and emotions.
- `top-10-cities.html` demonstrates interactive web mapping with markers, popups, base map choices, and a collapsible city information panel.
- `data-collection.html` explains a YouTube crawler workflow used to collect migration-related search results and visualize them through word clouds.
- `geospatial-classification.html` presents a church classification counter-map using OSM-based categories of belonging and unfamiliarity.
- `styles.css` provides the shared visual design for the website.

## Core Themes

This repository deals with:

- qualitative and humanistic GIS
- participatory and critical mapping
- geospatial data collection
- classification as interpretation
- migration, belonging, and social difference
- web-based geospatial storytelling

The project treats GIS as more than a technical mapping tool. It asks how categories, interfaces, data sources, and visual choices shape what becomes visible, meaningful, or hidden.
